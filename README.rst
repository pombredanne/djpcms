=========
Djpcms
=========

A dynamic content management system for Django and jQuery

Required
-----------

 * Python >= 2.5
 * django >= 1.1.1

Optional
-----------
 * django-tagging	http://code.google.com/p/django-tagging/
 * django_compressor 
 

Running tests
---------------
On the top level directory type

python testing.py test djpcms
