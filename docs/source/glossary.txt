.. _glossary:

==========
Glossary
==========

.. glossary::
		
    plugin
		A class responsible for rendering applications and static content across any page
		in the web site.
		
		See :ref:`plugins <plugins-index>`.
		
    template
        A chunk of text that acts as formatting for representing data. A
        template helps to abstract the presentation of data from the data
        itself.
        
        See :ref:`topics-templates-index`.
        
    view
        A class responsible for handling requests. Each view class
        is associated with a url, or set of urls. Each view can have several
        response method, post, get, etc ...
        
        See :ref:`views handlers <views-index>`.
