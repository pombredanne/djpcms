.. _glossary:

========
Glossary
========

.. glossary::
		
    plugin
		A class responsible for rendering applications and static content.
		
		See :ref:`plugins-index`.
		
    template
        A chunk of text that acts as formatting for representing data. A
        template helps to abstract the presentation of data from the data
        itself.
        
        See :ref:`topics-templates-index`.
        
    view
        A class responsible for rending a page.
        
        See :ref:`views-index`.
