============================
Uniform Utility
============================

.. module:: djpcms.utils.html.uniform

This module complements the django forms library by adding Ajax funationality, inline formes,
and custom layout based on uniform_ style.

This module is easy to use::

	class MyForm(forms.Form);
		name = forms.cahrField()
		
		helper = formHelper()
